# Ecommerce React Website

> Full-featured frontend e-commerce application built with React, CSS, and Hooks.

[![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![React](https://img.shields.io/badge/React-61DAFB?style=flat-square&logo=react&logoColor=black)](https://react.dev/)

## Overview

A complete e-commerce frontend with product listing, cart management, and checkout flow. Built using modern React patterns — functional components, Hooks for state management, and CSS for styling — with no external UI libraries.

## Features

- Product catalog with listing and detail views
- Shopping cart — add, remove, update quantity
- State management via React Hooks (`useState`, `useEffect`, `useContext`)
- Responsive layout

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React |
| Language | JavaScript |
| State | React Hooks (useState, useContext) |
| Styling | CSS Modules |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/vfiedlerc/ecommerce-react-website-app.git
cd ecommerce-react-website-app

# Install dependencies
npm install

# Run locally
npm start
```

App runs at `http://localhost:3000`

## Project Structure

```
src/
├── components/       # Reusable UI components
├── pages/            # Route-level views (Home, Product, Cart)
├── context/          # Global state via Context API
├── hooks/            # Custom hooks
└── styles/           # CSS modules
```
